## Source of truth

All Protobuf schemas live in `proto/`.

- Buf module configuration: `buf.yaml`
- Dependency lockfile: `buf.lock` (generated via `buf dep update`)

## Dependencies

This repository uses:

- Google API field behaviors: `googleapis/googleapis`
- Validation annotations: `bufbuild/protovalidate`

## Swap API schema

The swap API schema is defined at:

- `proto/ln_liquid_swap/v1/swap.proto`

This API is intentionally gRPC-only (no HTTP JSON transcoding).
